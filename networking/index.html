This project serves as a showcase of cloud infrastructure, bridging custom domain identity with NGINX-powered web hosting on an Amazon EC2 instance. By configuring DNS to map to a public IPv4 address, the setup creates a direct, high-performance link to the cloud-based server. The result is a professional, live deployment illustrating the integration of IP routing and web services. You can read the full guide on setting up a web server on AWS.

Hosted with NGINX · Deployed on AWS EC2 · Custom domain via DNS A record.
